# 🐤 Flappy Bird Clone

A simple, classic web-based Flappy Bird clone built with HTML5 Canvas, CSS, and Vanilla JavaScript.

## ✨ Features
- **Classic Gameplay:** Press the Spacebar to flap and avoid the pipes.
- **Infinite Generation:** Pipes are procedurally generated for endless gameplay.
- **Score Tracking:** Keeps track of your score as you successfully pass through pipes.
- **Collision Detection:** Accurate hitboxes for the bird, pipes, floor, and ceiling.
- **Game State Management:** Handles starting, game over screens, and restarting seamlessly.

## 🛠️ Tech Stack
- **HTML5** (Canvas API)
- **CSS3** (Styling and gradient backgrounds)
- **Vanilla JavaScript** (Game loop using `requestAnimationFrame`, physics, and logic)

## 🚀 How to Run
1. Download or clone the project files.
2. Simply double-click the `index.html` file to open it in any modern web browser.
3. No local server, npm, or build tools required!

## 🎮 Controls
- **Spacebar:** Flap / Restart Game after Game Over.
